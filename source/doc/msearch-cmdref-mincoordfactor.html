MinCoordFactor

Name

MinCoordFactor -- gives more score to documents having query words closer to the beginning

search.htm

Synopsis

MinCoordFactor {number}

MinCoordFactor gives more score to the documents having the first found word closer to the beginning of the document.

Use with a number between 0 and 255.

The default value is 0, which means no effect.

Examples

MinCoordFactor 100
