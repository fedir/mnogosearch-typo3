HTDBLimit

Name

HTDBLimit -- limits the amount of document IDs fetched in a single HTDBList query

indexer.conf

Description

Using HTDBLimit and the other HTDB related commands is described in details in the Section called Indexing SQL tables (htdb:/ virtual URL scheme) in Chapter 4.

Scope

HTDBLimit affects all following Server commands, until the end of the configuration file, or until the next HTDBLimit command.

Examples

HTDBLimit 100000

See also

HTDBAddr, HTDBDoc, HTDBList, UserCacheQuery.
