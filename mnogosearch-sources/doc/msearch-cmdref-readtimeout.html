ReadTimeOut

Name

ReadTimeOut -- defines stalled connections timeout

indexer.conf

Synopsis

ReadTimeOut {time}

Description

ReadTimeOut defines connect timeout and stalled connections timeout.

See the Period command description for time format.

The default value is 30 seconds.

Scope

ReadTimeOut can be set multiple times and affect on all following Server and Realm command until the end of the configuration file, or until the next ReadTimeOut command.

Examples

ReadTimeOut 30s
