UseNumericOperators

Name

UseNumericOperators -- defines whether to interpret numeric operators in a search query

search.htm

Synopsis

UseNumericOperators {yes | no}

When UseNumericOperators is set to yes, the < and > signs are interpreted as numeric operators and ignored otherwise.

The default value is no.

Examples

UseNumericOperators yes
