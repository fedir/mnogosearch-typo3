HlBeg

Name

HlBeg -- defines left search results highlighting code

search.htm

Synopsis

HlBeg {string}

Description

Search query words are highlighted in search results using the given string to the left and the string given in the HlEnd command to the right.

Examples

HlBeg "<strong>"
HlEnd "</strong>"
