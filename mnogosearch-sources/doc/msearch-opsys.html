Supported operating systems

We use GNU Autoconf so it is possible to compile mnoGoSearch on almost any modern UNIX system with a C compiler. We develop mnoGoSearch using the FreeBSD and Linux operating systems.

mnoGoSearch has been reported to compile and work fine on the following operating systems:

  • FreeBSD 2.2.x, 3.x, 4.x, 5.x

  • Linux 2.x

  • sun4u sparc SunOS 5.6

  • Solaris7 x86

  • OpenSolaris 2008

  • BSDI 4.1

  • OpenBSD 2.5

  • HP-UX B.10.20 (GNU sed required)

  • Digital Unix V4.0 (aka Compaq Tru64 UNIX)

  • SCO UnixWare 7.x

  • AIX 4.2

  • SGI Irix 6.5

  • YellowDog Linux for PPC on Mac G3

  • Mac OS X 10.0.0, Mac OS X 10.2(.4)

  • Debian Linux for PPC on Mac G3

  • Debian Linux for PPC on Mac iBook

  • Suse Linux for PPC on Mac iBook

We hope mnoGoSearch will compile and work on other UNIX platforms as well. Please report successful platforms to devel@mnogosearch.org.
