HlEnd

Name

HlEnd -- defines right search results highlighting code

search.htm

Synopsis

HlEnd {string}

Description

Search query words are highlighted in search results using the HlBeg value to the left and the HLEnd value to the right.

The values of HlBeg and HTEnd typically consist of some HTML code changing font color or font style.

Examples

HlBeg "<strong>"
HlEnd "</strong>"
