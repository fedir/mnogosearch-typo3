Creating a binary package

You can create a binary distribution for your platform and share it with us. We'll put the contributed binaries for various platforms to our site to make them available for other users. To create a binary distribution, please type make bin-dist, then put the newly created tar.gz file somewhere on the Internet and send an e-mail to <general@mnogosearch.org> (in English) or to <ru@mnogosearch.org> (in Russian).
