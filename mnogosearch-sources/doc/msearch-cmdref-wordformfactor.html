WordFormFactor

Name

WordFormFactor -- gives more score to the original query word form (as opposite to Synonym or Ispell fuzzy forms)

search.htm

Synopsis

WordFormFactor {number}

Description

WordFormFactor gives more score to the original word forms written in a search query, and to give less scores to fuzzy word forms generated using Synonyms or Ispell dictionaries.

The number should be in the range 0..255, with lower values meaning stronger effect.

The default value is 255, which means to give the same weight to the original and the fuzzy forms. 0 means maximum effect, i.e. weight for the generated word forms will be much smaller than weight for the original word form.

This command appeared in mnoGoSearch version 3.3.0.

Examples

WordFormFactor 100
