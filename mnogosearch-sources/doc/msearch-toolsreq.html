Tools required for installation

You need the following tools to build and install mnoGoSearch from sources:

  • GNU gunzip to uncompress the distribution.

  • A reasonable tar to unpack the distribution. GNU tar is known to work.

  • A working ANSI C compiler. GNU gcc is known to work.

  • A good make program. GNU make is recommended and sometimes is required.

  • sed - stream editor.

  • Optionally: Perl interpreter, if you're going to use install.pl (a helper script).

  • Optionally: jade or openjade to build the manual from the XML sources,
