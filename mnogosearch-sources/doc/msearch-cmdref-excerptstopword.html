ExcerptStopword

Name

ExcerptStopword -- defines whether to highlight stopwords.

search.htm

Synopsis

ExcerptStopword {yes | no}

Description

ExcerptStopword defines if search.cgi should highlight stopwords in excerpts.

The default value is yes.

Examples

ExcerptStopword no
